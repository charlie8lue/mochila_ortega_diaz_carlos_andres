¿Quién inventó el sistema de control de versión Git y por qué?

• Fue inventado por Linus Torvalds debido a que los sistemas de control de versiones de la época no cumplían a cabalidad con sus necesidades y además, eran de carácter privado.

¿A quién pertenece actualmente Github y por qué?

• Github fue comprado por Microsoft el 4 de junio del 2018. En mi opinión lo hace como una estrategia para acercarce a los desarrolladores del mundo debido a la popularidad de Github, e impulsar sus productos. Cómo sucedió en su momento con Visual Studio Code, un salto enorme por parte de Microsoft al experimentar en el Open Source.

¿Hay otra forma que no sea la terminal para trabajar con Github?

Github también puede ser utilizado en Visual Studio Code con la instalación de una extensión de Git.
